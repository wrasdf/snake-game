## Why

A classic snake game provides an engaging, interactive web experience that demonstrates fundamental game mechanics. This is an ideal starting point for web game development, combining simple controls with satisfying gameplay loops (eat, grow, survive).

## What Changes

- New web-based snake game with real-time gameplay
- 4-directional keyboard controls (arrow keys) to move the snake
- Random food blocks that spawn on the game board
- Snake grows in length each time it eats food
- Game board with collision detection (walls and self-collision end the game)
- Real-time score tracking based on food consumed
- Clean, responsive UI for gameplay display

## Capabilities

### New Capabilities
- `snake-movement`: Control snake direction using 4 arrow keys (up, down, left, right)
- `food-spawning`: Randomly generate food blocks on the game board
- `snake-growth`: Extend snake length when eating food
- `collision-detection`: Detect wall and self-collision to end the game
- `game-ui`: Display game board, score, and game-over state

- Creates new web application with HTML/CSS/JavaScript
- Requires game loop and rendering system
